# AgAI_17: Autonomous Agent Platform

A Manus-style autonomous AI agent platform for Datawebify. Users submit a natural language goal and the system plans and executes it end-to-end without manual intervention.

## Stack
- Frontend: Next.js 14 + TypeScript + Tailwind CSS
- Backend: FastAPI + Python 3.12 + LangGraph
- Database: Supabase (PostgreSQL)
- Orchestration: LangGraph + Specialist Agents
- Deployment: Vercel (frontend) + Railway (backend)
